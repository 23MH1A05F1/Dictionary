

# Dynamic Dictionary Web App

A fully functional, responsive dictionary web application built with HTML, CSS, and JavaScript. Search any word and get instant definitions, pronunciation audio, and part-of-speech information using the Dictionary API.

<img width="888" height="849" alt="image" src="https://github.com/user-attachments/assets/5c2b1454-8d20-4510-8739-c2861bd14392" />

##  Features

-  **Real-time word search** with Dictionary API integration
-  **Audio pronunciation** with one-click playback
-  **Fully responsive** design (mobile & desktop)
-  **Modern UI** with smooth animations and transitions
-  **Input validation** and clear search functionality
-  **Fast loading** with optimized CSS and CDN resources

##  Tech Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **API**: [Dictionary API](https://dictionaryapi.dev/)
- **Icons**: Font Awesome 5
- **Fonts**: Google Fonts (Open Sans)
- **Responsive**: CSS Media Queries

##  Prerequisites

- Basic knowledge of HTML, CSS, and JavaScript
- Web browser (Chrome, Firefox, Safari, Edge)
- Code editor (VS Code recommended)

## Troubleshooting
| Issue                | Solution                                      |
| -------------------- | --------------------------------------------- |
| No definitions found | Check internet connection, try different word |
| Audio not playing    | Ensure browser allows autoplay                |
| Styles not loading   | Check Font Awesome CDN connection             |
| API errors           | Verify API endpoint in script.js              |
